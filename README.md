# Technician Finding Application
## Client Site
> cd client
#### Set .env file
> cp .env.example .env
>> Add GOOGLE_API in .env file

##### For IOS
```
  1. yarn install or npm install
  2. cd ios && pod install
  3. cd ..
  4. react-native run-ios
```
##### For Android
```
  1. yarn install or npm install
  2. react-native run-android
```



### Server Site
> cd server
```
  1. yarn install or npm install 
  2. cp อีเอ็นวี.txt .env
  3. npm start
```
